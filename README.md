# Personal Portfolio Website

A responsive personal portfolio website showcasing my projects, skills, education, and contact information.

## Live Demo

Visit the live website:
https://portfolio-eight-rust-77.vercel.app

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and clean user interface
* About Me section
* Skills showcase
* Project portfolio with live demos and source code links
* Contact section
* Smooth navigation and animations

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Vercel (Deployment)

## Projects Included

### Portfolio Website

A personal portfolio website designed to showcase my work, skills, and experience.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Nilaxi11/Portfolio.git
```

2. Navigate to the project directory:

```bash
cd Portfolio
```

3. Open `index.html` in your browser.
